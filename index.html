KitX

万物互联的插件扩展平台

项目简介 快速上手 (桌面端) 快速上手 (移动端)

架构优雅

优秀的设计使得 KitX 拥有极强的稳定性, 且易于扩展。

万物互联 (桌面端)

在你的不同设备间流转数据, 搭建工作流, 就像在一台设备上一样。

万物互联 (移动端)

用户可以在手机上控制整个 KitX 网络, 掌握自己的工作流, 例如遥控自己的电脑, 执行自动化任务.

更多数据

手机上的各类传感器将提供更多数据, 供用户的工作流使用.

跨平台体验 (桌面端)

享受 .NET 带来的跨平台基础, 体验优秀的跨平台 UI 框架 Avalonia。

跨平台体验 (移动端)

享受 Flutter 带来的跨平台基础, 体验优秀的跨平台 UI 框架 Material Design.
